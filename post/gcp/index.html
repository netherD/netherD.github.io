v2ray相关

log日志/var/log/v2ray

查看配置文件 /etc/v2ray/config.json，里面有一段类似于如下的内容：

1
2
3
4
5

"log": {
  "access": "访问日志文件地址",
  "error": "错误日志文件地址",
  "loglevel": "warning"
}

把 access 或是 error 设为空字符串，就可以不启用对应的那一项日志。
